# Tableau de Bord Routeur

Ny tableau de bord (`/dashboard`) dia manome fijery fiasana voatambatra.

## Inona no arahina maso

- sata routeur an-tserasera/tsy an-tserasera,
- fironana sessions sy varotra,
- fampitandremana fiasana,
- mari-pamantarana fahasalaman'ny sehatra ankapobeny.

## Raha efa tranainy ny data

1. Jereo ny scheduler sy queue worker.
2. Jereo ny fampifandraisana API routeur.
3. Diniho ny logs rindranasa sy Nginx/PHP-FPM.
