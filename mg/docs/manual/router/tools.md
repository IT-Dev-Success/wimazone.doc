# Fitaovana Rafitra

Ny fitaovana fiasana dia mikasika ny diagnostika rindranasa sy routeur.

## Fitaovana mahazatra

- fitsapana fampifandraisana routeur,
- sync routeur/hotspot manual,
- logs sy fitsirihana fahasalamana,
- pejy fitaovana fiasana/admin.

## Zavatra arahina maso

- fiakaran'ny timeout `/hotspot/status`,
- PHP-FPM `max_children` / `SIGKILL`,
- hadisoana Nginx `upstream timed out` na `502`.
