# Fanontana Haingana

Ny Fanontana Haingana dia ampiasaina amin'ny fanontana tikety voucher avy amin'ny dingana voucher.

## Fepetra takiana

1. Modely tikety mavitrika manan-kery.
2. Voucher misy aorian'ny sivana.
3. Alalana hiditra amin'ny module voucher.

## Raha tsy mety ny fanehoana

- hamarino ny modely/override mavitrika,
- jereo ny variables modely takiana,
- tsapao amin'ny modely fallback kely indrindra.
