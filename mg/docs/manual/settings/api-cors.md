# API & CORS

Ity fizarana ity dia mikasika ny fampifandraisana ao amin'ny **Paramètre > API** (`/settings/apis`).

## CORS

Apetraho mazava ny origins azo ekena:

1. Ampio origins marina (`https://example.com`).
2. Soroka ny wildcard `*` amin'ny production.
3. Avelo ihany ny fomba ilaina.
4. Hamarino amin'ny devtools navigateur aorian'ny fanavaozana.

## API Ivelany

Arakaraka ny modules alefaina:

- MVola
- SMTP
- Tawk.to
- configuration ivelany hafa

## Fiarovana

- Tehirizo ny tsiambaratelo amin'ny lafin'ny serveur ihany.
- Ampiasao porofon-teny samihafa isaky ny tontolo iainana.
- Ferao ny paramètre API amin'ny `Super Admin`.
