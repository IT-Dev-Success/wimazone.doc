# Paramètre Rafitra

Ny paramètre rafitra dia mikasika ny safidy fitantanana sehatra manontolo.

## Faritra mahazatra

- profil admin (`/profile`)
- paramètre ankapobeny (`/settings/general`)
- loko theme (`/settings/colors`)
- fampandrenesana admin

## Fiarovana

- teny miafina matanjaka sy fanovana tsy tapaka,
- alalana kely indrindra ho an'ny mpampiasa admin,
- akatony ny sessions amin'ny fitaovana zaraina,
- diniho ny andraikitra/alalana tsy tapaka.
