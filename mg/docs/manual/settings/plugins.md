# Plugins Portail Mpanjifa

Ny plugins portail mpanjifa dia tantanina ao amin'ny:

- `/settings/customer-portal/plugins`

## Asa

- Mametraka plugin ZIP
- Alefaso/Ajanony
- Sync paramètre
- Manavao plugin
- Mamafa plugin
- Mijery plugin mialoha

## Soso-kevitra

- Apetraho plugins azo itokisana ihany.
- Hamarino ny fitoviana amin'ny version tetikasa ankehitriny.
- Jereo indray ny fanehoana amin'ny finday sy ordinateur aorian'ny fampiasana.
