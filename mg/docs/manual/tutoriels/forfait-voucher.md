# Mamorona forfait sy mamokatra voucher

Ity tutoriel ity dia mikasika ny famoronana forfait internet sy fanamboarana tikety voucher vonona hamidy.

## <Icon name="Package" color="info" /> 1) Mamorona forfait internet

1. Menu → **Facturation & Forfaits → Forfaits Internet**
2. Kitio **+ Forfait vaovao**
3. Fenoy :

| Saha | Ohatra |
|---|---|
| **Anarana** | `Indray andro 1 GB` |
| **Faharetana** | `24 ora` |
| **Habe** | `1024 MB` (0 raha tsy misy fetra) |
| **Bande passante** | `2M/2M` (download/upload) |
| **Vidiny (Ar)** | `1000` |
| **Profil MikroTik** | `1day-1gb` (hovokarina avy hatrany) |
| **Mavitrika** | ✓ |

4. **Tehirizo**. Alefan'ny Wima Zone amin'ny routeur ny profil.

## <Icon name="Ticket" color="success" /> 2) Mamokatra voucher

1. Menu → **Gestion des clients → Tickets**
2. Kitio **+ Mamokatra voucher**
3. Fenoy :

| Saha | Ohatra |
|---|---|
| **Forfait** | `Indray andro 1 GB` |
| **Isa** | `50` |
| **Halaven'ny code** | `8` litera |
| **Endrika** | Alphanumeric majuscule |
| **Daty fahataperana** | *(avela foana raha tsy misy fetra)* |

4. Kitio **Vokareo**. Vokarina indray mandeha ny 50 code.

## <Icon name="Printer" color="primary" /> 3) Pirinty ny tikety

1. Tsongay ny voucher (case à cocher)
2. Kitio **Pirinty**
3. Fidio ny template (oh. `Tikety 58mm thermique` na `A4 – 10 isan-dtakelaka`)
4. Miseho ny aperçu. Kitio **Pirinty**.

::: tip Pirinty thermique
Ho an'ny pirinty 58mm mifandray amin'ny Android/POS, ampiasao ny template **Tikety 58mm** ao amin'ny Paramètres → Templates tikety.
:::

## <Icon name="BarChart" color="warning" /> 4) Arao ny stock

Ao amin'ny **Dashboard → Stock Tikety**, hitanao foana :

- `Tsy ampiasaina` : voucher mbola tsy namidy
- `Ampiasaina` : voucher efa nampiasain'ny mpanjifa

Raha latsaka 20% ny stock, miseho ny alerte volomboasary.
