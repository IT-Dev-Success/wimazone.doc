# Zahao ny tatitra

Manome tatitra mivondrona ny Wima Zone ho an'ny fitantanana ny orinasa hotspot.

## <Icon name="BarChart" color="info" /> 1) Tatitra misy

Menu → **Rapports & Analytique → Rapports**

Lisitra :

- **Vola miditra araka ny fotoana** — isan'andro, isan-kerinandro, isam-bolana
- **Varotra araka ny forfait** — forfait be mpividy
- **Varotra araka ny routeur** — performance isaky ny toerana
- **Fanombohana vs fahataperana** — taha fampiasana
- **Top mpanjifa**
- **Tatitra Cafe** — raha alefa ny module Cyber Café

## <Icon name="Calendar" color="primary" /> 2) Filtre araka ny fotoana

- **Androany** / **Omaly**
- **7 andro farany** / **30 andro farany**
- **Ity volana ity** / **Volana lasa**
- **Fotoana manokana**

## <Icon name="PieChart" color="warning" /> 3) Sary

Ny tatitra rehetra dia misy :

- **Graphika** (barres, courbes, camembert)
- **Tabilao** feno
- **Total** (vola, isa, taha)

## <Icon name="Download" color="success" /> 4) Fanondranana

- **CSV** — andalana rehetra
- **PDF** — ho an'ny pirinty
- **Pirinty** mivantana

## <Icon name="FileText" color="primary" /> 5) Tatitra ho azy

Ao amin'ny **Paramètres → Tatitra ho azy**, azo programmé :

- Isan'andro → mailaka amin'ny 23h00
- Isan-kerinandro → alatsinainy maraina
- Isam-bolana → andro voalohan'ny volana

Lasa PDF amin'ny mailaka.

## <Icon name="TrendingUp" color="info" /> 6) Indicateur lehibe (dashboard)

Amin'ny **Dashboard** :

| Karatra | Votoaty |
|---|---|
| **Vola androany** | Total Ar |
| **Vola ity volana ity** | Cumul |
| **Mavitrika / Lany** | Ratio voucher |
| **Mpanjifa** | Isa total |
| **Tikety** | Stock sisa |

## <Icon name="Database" color="warning" /> 7) Archivage

Ho an'ny database malama :

- Menu → **Paramètres → Rafitra → Archivage**
- Ovay ny seuil (oh. > 12 volana)
- **Archivage izao**
