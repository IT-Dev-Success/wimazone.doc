# Tutoriel

Ity fizarana ity dia ahitana **dingana tsirairay** ho an'ny asa mahazatra, aorian'ny fametrahana ny Wima Zone Billing amin'ny MikroTik.

## <Icon name="Compass" color="info" /> Alohan'ny hanombohana

Hamarino fa :

- Mandeha ny container (`/container/print` → `status: running`)
- Afaka miditra amin'ny portail admin ianao (`http://172.17.0.2` na ny port 8080)
- Manana rôle **Super Admin** na alalana mifanaraka
