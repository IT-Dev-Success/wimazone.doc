# Logos and Branding

Branding impacts:

- admin interface,
- customer portal,
- printed tickets.

## Recommended formats

- main logo: SVG or transparent PNG,
- secondary variants: optimized PNG/JPG,
- keep file size reasonable for fast loading.

## Quality checks

1. Verify desktop/mobile rendering.
2. Verify contrast on light and dark backgrounds.
3. Verify ticket print output after logo updates.
