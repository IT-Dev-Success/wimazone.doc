# User Manual

This manual covers daily operations in **WimaZone Billing** for hotspot administration.

## Prerequisites

- Signed in with an admin account.
- Proper permissions (routers, hotspot, vouchers, settings, reports).
- At least one active MikroTik router configured.

## Sections

### [Global Settings](/en/docs/manual/settings/)

- Router management and API connectivity tests.
- API/CORS, SMTP, SMS, and payment configuration.
- Customer portal plugins and ticket print templates.
- Branding (logos, colors, system settings).

### [Router Operations](/en/docs/manual/router/)

- Dashboard and operational health checks.
- Hotspot management (provisioning, walled garden, IP bindings, snapshots).
- Voucher printing and field operations.
- Reports by date, plan, router, and exports.
