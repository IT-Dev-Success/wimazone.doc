# System Tools

Operational tools cover app and router diagnostics.

## Common tools

- router connectivity tests,
- manual router/hotspot sync,
- logs and health checks,
- operations/admin utility pages.

## Watch for

- `/hotspot/status` timeout spikes,
- PHP-FPM `max_children` / `SIGKILL`,
- Nginx `upstream timed out` or `502` errors.
