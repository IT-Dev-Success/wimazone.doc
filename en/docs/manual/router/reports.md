# Reports

Reports are available in `/reports`.

## Available views

- overview (`/reports`)
- by plan (`/reports/by-plan`)
- by router (`/reports/by-router`)
- date range (`/reports/date-range`)
- exports (`/reports/export`, `/reports/export-accounting`)

## Recommended flow

1. Set analysis period.
2. Filter by router/plan.
3. Compare trends before/after network or pricing changes.
4. Export results for audit.
