# Manuel utilisateur

Ce manuel décrit l'utilisation quotidienne de **Wima Zone Billing** côté administration hotspot.

## Prérequis

- Être connecté avec un compte admin.
- Disposer des permissions adaptées (routeurs, hotspot, vouchers, paramètres, rapports).
- Avoir au moins un routeur MikroTik configuré et actif.

## Rubriques

### [Paramètres globaux](/docs/manual/settings/)

- Gestion des routeurs et test de connexion API.
- Configuration API/CORS, SMTP, SMS, paiements.
- Templates d'impression ticket.
- Branding (logos, couleurs, paramètres système).

### [Opérations routeur](/docs/manual/router/)

- Supervision dashboard et santé opérationnelle.
- Gestion hotspot (provisioning, Walled Garden, IP bindings, snapshots).
- Impression vouchers et exploitation terrain.
- Rapports par date, forfait, routeur et exports.

### [Tutoriels](/docs/manual/tutoriels/)

- Première connexion et configuration initiale (branding, comptes admin).
- Configurer les APIs externes (MVola, SMS, SMTP).
- Créer un forfait et générer des vouchers.
- Suivre les sessions actives et consulter les rapports.

### [Module Cyber Cafe](/docs/manual/cafe/)

- Gestion des postes de travail (creation, verrouillage, suppression).
- Sessions prepayees et postpayees avec facturation automatique.
- Mode libre-service pour les clients.
- Controle d'acces reseau MikroTik (blocage/deblocage automatique).
- Client Flutter desktop avec ecran de verrouillage et code d'activation segmente.
- [Configuration MikroTik pour le cafe](/docs/manual/cafe/mikrotik)
- [Client Flutter](/docs/manual/cafe/client)
