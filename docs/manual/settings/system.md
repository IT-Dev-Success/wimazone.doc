# Paramètres système

Les paramètres système regroupent les options globales d'administration.

## Éléments fréquents

- profil administrateur (`/profile`)
- paramètres généraux (`/settings/general`)
- couleurs thème (`/settings/colors`)
- notifications admin

## Langues et affichage

- langues docs/interface supportées: Français / English
- valider le rendu des thèmes après modification des couleurs

## Sécurité opérationnelle

- mots de passe forts et rotation régulière,
- moindre privilège pour les comptes admin,
- fermeture des sessions sur postes partagés,
- revue périodique des permissions rôles/utilisateurs.
