# Logos et branding

Le branding (logo, couleurs, identité visuelle) impacte:

- l'interface admin,
- le portail client,
- les tickets imprimés.

## Recommandations de format

- logo principal: SVG ou PNG transparent,
- variantes secondaires: PNG/JPG optimisés,
- taille raisonnable pour éviter les lenteurs de chargement.

## Contrôle qualité

1. Vérifier rendu desktop/mobile.
2. Vérifier contraste texte/fond.
3. Vérifier impression ticket après remplacement logo.
