# Plugins portail client

Les plugins du portail client sont gérés dans:

- `/settings/customer-portal/plugins`

## Opérations disponibles

- Installer un plugin ZIP
- Activer/désactiver
- Synchroniser les paramètres
- Rafraîchir un plugin
- Supprimer un plugin
- Prévisualiser un plugin

## Bon format de plugin

- Archive ZIP propre (structure attendue par l'application)
- Assets statiques versionnés
- Paramètres avec valeurs par défaut

## Recommandations

- Installer uniquement des plugins de confiance.
- Valider sur un environnement de test avant production.
- Refaire une vérification visuelle mobile/desktop après activation.
