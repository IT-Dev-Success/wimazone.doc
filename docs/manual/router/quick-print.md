# Impression rapide (Quick Print)

L'impression rapide concerne la production de tickets vouchers depuis les écrans vouchers.

## Pré-requis

1. Un template ticket valide et actif.
2. Vouchers disponibles selon les filtres.
3. Permission d'accès au module vouchers.

## Scénarios

- impression unitaire,
- impression multiple,
- aperçu avant impression (`print-preview`).

## Si le rendu est incorrect

- vérifier le template actif/override,
- vérifier les variables attendues dans le template,
- tester avec un template de secours minimal.
