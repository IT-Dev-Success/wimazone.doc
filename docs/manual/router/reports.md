# Rapports

Les rapports sont disponibles dans `/reports`.

## Vues disponibles

- général (`/reports`)
- par forfait (`/reports/by-plan`)
- par routeur (`/reports/by-router`)
- plage de dates (`/reports/date-range`)
- exports (`/reports/export`, `/reports/export-accounting`)

## Méthode recommandée

1. Définir la période d'analyse.
2. Filtrer par routeur/forfait.
3. Comparer avant/après changements réseau ou tarifaires.
4. Exporter le résultat pour audit.

## Diagnostic

Recouper les rapports avec:

- logs d'activation vouchers,
- sessions actives,
- incidents routeur/hotspot.
