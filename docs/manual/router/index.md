## Dépendances

Les actions routeur nécessitent:

- routeur actif et joignable,
- identifiants API valides,
- permissions applicatives adaptées.

## Incident le plus courant

`Invalid user name or password` signifie généralement:

- mauvais identifiant API,
- compte API désactivé,
- droits insuffisants côté MikroTik.
